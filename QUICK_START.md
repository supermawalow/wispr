# ⚡ WHISPR PRO - Быстрый старт

## 🚀 Запуск за 2 минуты

### Шаг 1: Сервер
```bash
cd whispr-pro/server
npm install
npm start
```
✅ Сервер работает на http://localhost:3001

### Шаг 2: Клиент (новое окно терминала!)
```bash
cd whispr-pro/client
npm install
npm run dev
```
✅ Открой http://localhost:3000

## 👑 Админ вход

Уже создан аккаунт администратора:
- **Username:** `admin`
- **Password:** `admin123`

Войди и получи доступ к админ-панели!

## 📝 Что можно делать?

### Как обычный пользователь:
1. ✅ Зарегистрироваться
2. ✅ Искать других пользователей
3. ✅ Добавлять в контакты
4. ✅ Писать личные сообщения
5. ✅ Видеть кто онлайн

### Как администратор:
1. ✅ Просмотр статистики
2. ✅ Список всех пользователей
3. ✅ Удаление пользователей
4. ✅ Назначение других админов
5. ✅ Просмотр онлайн-статуса всех

## 🎯 Быстрый тест

### Вариант 1: Открой 2 вкладки браузера

**Вкладка 1:**
1. Зарегистрируйся (username: `alice`, пароль: `123`)

**Вкладка 2:**
1. Зарегистрируйся (username: `bob`, пароль: `123`)
2. Найди `alice` через поиск
3. Добавь в контакты
4. Открой чат с Alice
5. Пиши сообщения!

**Вкладка 1:**
1. Найди `bob`, добавь в контакты
2. Открой чат
3. Сообщения синхронизируются! ✨

### Вариант 2: Тест админки

1. Войди как `admin` / `admin123`
2. Нажми иконку щита (Shield) вверху справа
3. Посмотри статистику
4. Увидишь всех зарегистрированных пользователей

## 🎨 Основные функции

| Кнопка | Что делает |
|--------|-----------|
| 🔍 Найти пользователей | Открывает поиск |
| ➕ Добавить | Добавляет пользователя в контакты |
| ❌ (на контакте) | Удаляет из контактов |
| 🛡️ (вверху) | Админ-панель (только для админов) |
| 🚪 (вверху) | Выход |
| 👑 (в админке) | Сделать админом |
| 🗑️ (в админке) | Удалить пользователя |

## 💡 Полезные советы

✅ **Username** - только латиница, без пробелов  
✅ **Имя для отображения** - любое, как тебя будут видеть другие  
✅ **Поиск** - работает по username И по имени  
✅ **Онлайн** - зелёная точка = пользователь онлайн  
✅ **Печатает** - видно когда собеседник печатает  

## 🔥 Отличия от базового Whispr

**Базовый Whispr:**
- Общий чат для всех
- Нет регистрации
- Нет контактов

**Whispr Pro:**
- ✅ Личные чаты 1-на-1
- ✅ Система регистрации
- ✅ Контакты с поиском
- ✅ Админ-панель
- ✅ Username система

## 🆘 Проблемы?

**"Username уже занят"**  
→ Выбери другой username

**"Не могу найти пользователя"**  
→ Пользователь должен быть зарегистрирован

**"Сообщения не отправляются"**  
→ Проверь что сервер запущен (порт 3001)

**"Пустой список контактов"**  
→ Сначала найди и добавь пользователей!
